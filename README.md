# 🎯 Quest Tasks — Notion Reminder

เปลี่ยนสิ่งที่ต้องทำใน Notion ให้กลายเป็น **quest ที่อยากเคลียร์** พอถึงวันที่กำหนด
extension จะเด้งหน้าต่าง quest แบบเกมขึ้นมา เคลียร์เสร็จได้ XP สะสมเลเวลและ streak

---

## ติดตั้ง (Load unpacked)

1. เปิด `chrome://extensions`
2. เปิดสวิตช์ **Developer mode** (มุมขวาบน)
3. กด **Load unpacked** แล้วเลือกโฟลเดอร์ `quest-tasks` นี้
4. ปักหมุดไอคอน 🎯 ไว้บน toolbar จะใช้สะดวกขึ้น

---

## ตั้งค่าครั้งแรก

กดไอคอน → **เปิดหน้าตั้งค่า** แล้วทำ 3 ขั้น (ขั้น 4 เป็นของเมนู "อ่านทีหลัง" ทำตอนไหนก็ได้):

### ขั้น 1 — เชื่อมต่อ Notion
1. ไปที่ [notion.so/my-integrations](https://www.notion.so/my-integrations) → **New integration**
2. ตั้งชื่อ แล้วในส่วน **Capabilities** เปิดให้ครบทั้ง **Read content**, **Update content**, **Insert content**
3. คัดลอก **Internal Integration Secret** (`ntn_...`) มาวางในช่อง แล้วกด **ทดสอบ**

### ขั้น 2 — เตรียม database
1. เปิด page ใน Notion ที่จะใช้เก็บ (เช่นหน้า "งานของฉัน") → กด **•••** ขวาบน → **Connections** → เลือก integration ที่เพิ่งสร้าง
2. กลับมาที่หน้าตั้งค่า กด **โหลดรายการ** → เลือก page นั้นเป็นที่เก็บ → กด **สร้าง database**
3. เสร็จ! extension จะสร้าง database `🎯 Quest Tasks` ให้อัตโนมัติ พร้อม column: งาน, วันเตือน, เสร็จแล้ว, ระดับ

> มี database สำหรับ task อยู่แล้ว? สลับไปแท็บ **ใช้ database เดิม** แล้ววาง URL/ID ได้
> (แต่ชื่อ column ต้องตรงกับที่ระบบใช้: งาน / วันเตือน / เสร็จแล้ว / ระดับ)

หลัง create/link จะมีกล่อง **เช็คความพร้อม schema** โผล่ขึ้นมาอัตโนมัติ — เช็คว่า database มี column
ครบตามที่โค้ดเวอร์ชันปัจจุบันต้องใช้ไหม ถ้าขาดจะมีปุ่ม **"อัปเดต database (+N)"** ให้กด ถ้าครบแล้วปุ่มจะ
disable ขึ้น "เป็นปัจจุบันแล้ว ✓" ทุกครั้งที่สร้าง/เชื่อม/อัปเดต database (ไม่ใช่แค่ตอนอัปเดต) ระบบจะบันทึก
log ไว้ในหน้า Notion ("🛠 Migration Log") พร้อมเลข**เวอร์ชัน** — เปิด database นั้นแล้วเรียง column
"เวอร์ชัน" จากมากไปน้อยจะเห็นทันทีว่า database ไหนอยู่ที่เวอร์ชันล่าสุดหรือยัง (เปิดได้จากปุ่ม "ดู
migration log" ด้านล่างหน้าตั้งค่า)

### ขั้น 3 — การเตือน
ตั้งความถี่การเช็ค, เปิด/ปิดการเด้งหน้าต่าง quest อัตโนมัติ, และช่วงเวลาเงียบ (เช่น 22:00–07:00 ไม่ให้เด้งกวน)

### ขั้น 4 — อ่านทีหลัง (ทำตอนไหนก็ได้)
เลือก page แม่ (จะใช้ page เดียวกับขั้น 2 หรือเลือกใหม่ก็ได้) → กด **สร้าง database**
extension จะสร้าง database `📚 อ่านทีหลัง` แยกจาก quest ให้อัตโนมัติ

## ใช้งานประจำวัน

**เพิ่ม quest เร็ว** — กดไอคอน แล้วพิมพ์ในช่องด้านบน ระบบอ่านวันที่ภาษาไทยให้อัตโนมัติ:

| พิมพ์ | จะถูกตั้งวันเป็น |
|---|---|
| `โอนเงินสไมล์ พรุ่งนี้` | พรุ่งนี้ |
| `ประชุมทีม ศุกร์หน้า` | วันศุกร์ถัดไป |
| `จ่ายบิล สิ้นเดือน` | วันสุดท้ายของเดือน |
| `ส่งงาน อีก 3 วัน` | +3 วัน |
| `ซื้อของ 25/12` | 25 ธ.ค. |
| `เติมเงิน 5/7/2569` | 5 ก.ค. (แปลง พ.ศ. ให้) |
| (ไม่ระบุวัน) | วันนี้ |

**เคลียร์ quest** — พอถึงวันกำหนด หน้าต่าง quest จะเด้งขึ้นเอง (หรือกดไอคอนดูเองก็ได้)
กด **✓ เคลียร์** เพื่อ mark เสร็จ + รับ XP, หรือ **เลื่อน +1 วัน** ถ้ายังทำไม่ได้

**เพิ่มจากเว็บอื่น** — ลากเลือกข้อความบนหน้าเว็บไหนก็ได้ → คลิกขวา → **เพิ่มเป็น quest**

**ระบบเลเวล** — งานระดับ S ได้ XP เยอะสุด ไล่ลงมา A/B/C เคลียร์งานครบทุกวันติดกันจะสะสม streak 🔥

**อ่านทีหลัง** — เจอบทความ/ลิงก์ที่อยากอ่านแต่ยังไม่มีเวลา? เลือกข้อความหรือคลิกขวาที่ลิงก์ → **เก็บไว้อ่านทีหลัง**
แยก database จาก quest โดยสิ้นเชิง (ไม่มี due date, ไม่มี XP) สลับไปดูได้จากแถบเมนูด้านล่างของ popup
(กดแท็บ **📚 อ่านทีหลัง**) — สลับ tab ในหน้าเดียวกัน ไม่เปิดหน้าต่าง/แท็บใหม่

## แก้ปัญหาเบื้องต้น

| อาการ | สาเหตุ / วิธีแก้ |
|---|---|
| ทดสอบ token แล้วขึ้น error | คัดลอก token ผิด หรือยังไม่เปิด capability ครบ |
| โหลดรายการ page แล้วว่าง | ยังไม่ได้ share page ให้ integration — ทำขั้น 2 ข้อ 1 ก่อนแล้วกดโหลดใหม่ |
| สร้าง quest แล้ว error | integration ไม่มีสิทธิ์ Update/Insert — กลับไปเปิด capability ใน Notion |
| ไม่เด้งหน้าต่าง quest | เช็คว่าอยู่ในช่วง "เวลาเงียบ" หรือเปล่า / หน้าต่างเด้งวันละครั้ง |
| ขึ้นวันคลาดไป 1 วัน | ระบบยึด timezone กรุงเทพฯ อยู่แล้ว ถ้ายังเพี้ยนแจ้งได้ |

> **ความปลอดภัย:** token เก็บไว้ในเครื่องคุณเอง (`chrome.storage.local`) เท่านั้น ไม่ส่งไปไหน
> แต่เป็น plaintext — ถ้าจะแจกจ่าย extension ให้คนอื่น ต้องเปลี่ยนไปใช้ OAuth ก่อน (ดู roadmap)

## สิ่งที่ยังต้องทำต่อ (Roadmap)

สถานะปัจจุบัน: **v0.1 — ใช้งานได้ครบวงจรหลัก แต่ยังไม่ได้ทดสอบบน Chrome จริง**

ทำแล้ว ✅
- เชื่อม Notion + สร้าง database อัตโนมัติจากหน้าตั้งค่า
- quick-add พร้อม parser วันภาษาไทย (รวมแปลงปี พ.ศ.)
- เด้งหน้าต่าง quest แบบเกม + animation + เสียงตอนเคลียร์
- XP / level / streak, badge นับงานค้าง, ช่วงเวลาเงียบ
- เพิ่ม quest จากข้อความที่เลือกบนเว็บ (context menu)
- เมนู "อ่านทีหลัง" แยกจาก quest — เก็บลิงก์/ข้อความผ่าน context menu, database คนละตัว
- เช็ค & อัปเดต schema database อัตโนมัติ พร้อม migration log บน Notion (ยังไม่ทดสอบจริง)

ยังต้องทำ ⏳
- [ ] **ทดสอบจริงบน Chrome** ทุก flow (migrate, alarm เด้ง, เคลียร์, snooze)
- [ ] **งานทำซ้ำ (recurring)** เช่น "โอนค่าจ้างทุกสิ้นเดือน" ให้สร้างรอบถัดไปอัตโนมัติ
- [ ] **เลือกระดับ (S/A/B/C) ตอน quick-add** — ตอนนี้ตั้งเป็น B ให้ทุกอัน
- [ ] **retry เมื่อเน็ตหลุด** — เก็บงานที่บันทึกไม่สำเร็จไว้ลองใหม่
- [ ] **ปุ่ม Done/เลื่อน บน system notification**
- [ ] **รองรับงานค้างเกิน 100 รายการ** (pagination)
- [ ] **UI ปรับชื่อ column** สำหรับคนใช้ database เดิม
- [ ] **OAuth** ถ้าจะ publish ลง Chrome Web Store ให้คนอื่นใช้

รายละเอียดเชิงเทคนิคของแต่ละข้ออยู่ใน `CLAUDE.md`
